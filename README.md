# AndroidShowCase

The app use free music API. It has two main module:

- App: view layer
- Datasource: data layer

I used:

- SOLID principle
- common android best practice from the official documentation

##### Architecture:

- MVVM

##### Language:

- kotlin

##### Used library:

- RxJava
- Retrofit 2
- Glide
- Dagger 2
- canary leaks
- Android Architecture Components

##### Test lib

- Espresso
- Mockito
- JUnit
