# AndroidShowCase

The app uses free music API. It has two main modules:

- App: view layer
- Datasource: data layer

I used the followings:

- SOLID principles
- common android best practices from the official documentation

##### Architecture:

- MVVM

##### Language:

- kotlin

##### Used library:

- RxJava
- Retrofit 2
- Glide
- Dagger 2
- canary leaks
- Android Architecture Components

##### Test lib

- Espresso
- Mockito
- JUnit
